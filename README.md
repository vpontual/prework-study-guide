# Prework Study Guide Webpage

## Description

This Prework Study Guide was created for boot camp students who were going through the Prework. It contains notes on HTML, CSS, Git, and JavaScript.

## Installation

N/A

## Usage

Each section of this study guide contains notes about a certain topic, like HTML, CSS, JavaScript, Git. By opening the DevTools console, we can see the JavaScript running - It will list the topics we learned from prework and choose one topic for us to study first.
